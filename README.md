# CMS LP

CMS（Context Management System）商品のランディングページ。

- 商品仕様: `Cockpit/projects/consulting/CMS商品_LP仕様.md`
- 公開URL: https://ryeoungkim.github.io/cms-lp/
- 関連LP: [tw-lp](https://github.com/ryeoungkim/tw-lp)（CMS内オプション「Task Workflow」）

## 開発

```bash
npm install
npm run dev    # http://localhost:3000
npm run build  # 静的書き出し → out/
```

`main` への push で GitHub Pages へ自動デプロイ。
